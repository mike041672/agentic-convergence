# Agentic Convergence

Minimal starter for the Agentic AI Convergence engine.

## Goals
- Provide a reproducible repo structure for development and CI.
- Include a simple demo entrypoint and basic tests.
- Enable automated workflows and branch protection later.

## Quick start
```bash
# from project root
python -m venv .venv
.venv\Scripts\Activate.ps1   # PowerShell
pip install -r requirements.txt
python -m src.main
